TICKET ANALYSIS — RUN-1234

Summary
-------
Users report that the "Generate quarterly report" endpoint returns a 500 error
when the requested quarter has no transactions. Stack trace points to a
NullPointerException in QuarterlyReportService.buildSummary at line 142.

Reproduction
------------
1. Login as a user with no transactions in the target quarter
2. POST /api/reports/quarterly with quarter=Q1-2026
3. Observe HTTP 500 with empty body

Expected behavior
-----------------
The endpoint should return a 200 with an empty report structure (zero totals,
empty line items) per business rule REPORTING.RG042.

Stack trace
-----------
java.lang.NullPointerException: Cannot invoke "java.util.List.stream()" because "transactions" is null
  at com.example.reports.QuarterlyReportService.buildSummary(QuarterlyReportService.java:142)
  at com.example.reports.QuarterlyReportController.generate(QuarterlyReportController.java:58)
  at jdk.internal.reflect.GeneratedMethodAccessor93.invoke(Unknown Source)
  ...
